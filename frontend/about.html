📚 About This Project

Quantum Cryptography Tutorial is a comprehensive educational platform designed to help learners understand the principles of quantum computing and cryptography through interactive, hands-on tutorials.

It includes real-time simulations, visual explanations, and guided learning modules covering topics like quantum gates, qubits, circuits, and the BB84 protocol.

✨ Key Features

  • 🎯 Interactive quantum tutorials and visualizations
  • 🔐 BB84 protocol simulation with Alice, Bob, and Eve
  • 🧩 Drag-and-drop circuit builder with real-time feedback
  • 🌀 Dynamic Bloch sphere and state vector analysis
  • 💡 Educational error handling and explanations

🧰 Technology Stack

  • Frontend: HTML5, CSS3, Vanilla JavaScript
  • Backend: Python Flask
  • Quantum Toolkits: IBM Qiskit, QuTiP, Matplotlib

🎓 Learning Outcomes

After completing the tutorials, learners will be able to:

  • Understand the core concepts of quantum computing
  • Simulate and analyze quantum circuits
  • Explore the BB84 quantum key distribution protocol
  • Grasp the difference between classical and quantum logic
